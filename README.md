🧮 Simple Calculator
A command-line calculator built with Python, performing essential arithmetic and mathematical operations through a clean, menu-driven interface.
Features

Addition, subtraction, multiplication, and division
Power (exponentiation) and square root
Percentage calculation
Input validation (handles division by zero and negative square roots)

Technologies

Language: Python 3

Concepts: Functions, Conditionals, Loops, Exception Handling, math module

How to Run
bash python calculator.py
